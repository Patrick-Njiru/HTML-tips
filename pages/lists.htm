HTML Lists

  1. Display lowercase roman numbers by using the correct HTML attribute. 

                                <ol type="i">
                                <li> Item 1 </li>

                                <li> Item 2 </li>

                                <li> Item 3 </li>
                            </ol>
    1. Item 1
    2. Item 2
    3. Item 3

  2. Start counting from "30" by using the correct HTML attribute. 

                                <ol start="30">
                                <li> List Item </li>

                                <li> List Item </li>

                                <li> List Item </li>
                            </ol>
    1. List Item
    2. List Item
    3. List Item

  3. Detail Lists. 

                                <dl>
                                <dt> Tea </dt>
                                <dd> - hot beverage </dd>

                                <dt> Cocktail </dt>
                                <dd> - alcoholic mixed drink </dd>
                            </dl>
    Tea
    - hot beverage
    Cocktail
    - alcoholic mixed drink
