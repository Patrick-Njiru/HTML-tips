HTML Attributes

  1. attribute for making the content of an element editable.

    <p contenteditable="true"> Hi there. You can change me </p;>

    Hi there. You can change me

  2. boolean attribute to specify that an <input> tag must automatically get focus when the page loads.

    <input placeholder="Enter a text" autofocus>

  3. attribute to specify that the <button> is disabled.

    <button disabled> Click Me </button;>

  4. add a "tooltip" to a link, which must say "Go to homepage".

    <a href="../index.htm" title="Go to homepage"> Homepage </a>

    Homepage
